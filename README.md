⬛ A portfolio website that allows you to track historical transactions, their worth and destinations or sources.

⬛ A database management system written in FireBase with Google Authentication.

⬛ With the help of Coin Gecko API one can monitor price changes at all hours of the day and track your investments in realtime.

⬛The UI is made from Material UI which provides a simple, customizable and accessible library of React components.
